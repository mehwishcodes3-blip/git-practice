# 🌐 Language Switcher App (Flask)

This is a simple Flask web application that switches content between English and Urdu using URL parameters.

---

## 🚀 What it does

- Shows English or Urdu message based on URL
- Uses simple Flask routing
- Beginner-friendly project for learning web development

---

## 🧠 How it works

- `/` → default English message  
- `/?lang=en` → English message  
- `/?lang=ur` → Urdu message  

---

## 🛠️ Tech Stack

- Python
- Flask
